# Models Folder

Local cache for model weights used by Ayase modules.

Models are downloaded and cached automatically on first use via HuggingFace Hub, Torch Hub, and PyIQA. No manual setup is needed.

See [MODELS.md](../MODELS.md) for the complete inventory of all pretrained weights.

## Notes

- `models/` is git-ignored by default.
- Large model files should not be stored under `src/`.
